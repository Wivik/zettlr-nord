# zettlr-nord

A custom CSS for Zettlr using Nord theme palette

Prerequisite : https://www.fontsquirrel.com/fonts/droid-serif

How to : 

Open ~/.config/Zettlr/custom.css

Paste the content inside.

Enjoy.
